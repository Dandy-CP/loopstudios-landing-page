## Run Locally

Clone the project

```bash
  git clone https://github.com/Dandy-CP/loopstudios-landing-page.git
```

Go to the project directory

```bash
  cd loopstudios-landing-page
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
